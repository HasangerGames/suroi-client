# Suroi Client

The new client for the 2D battle royale Suroi, written in C++.

## Prerequisites
- C++20 or greater
- CMake

## Setup
```shell
git clone --recurse-submodules https://github.com/HasangerGames/suroi-client.git
cd suroi-client
mkdir build && cd build
cmake ..
cmake --build .
./suroi_client
```
